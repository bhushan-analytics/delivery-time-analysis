# Delivery Time Analysis 📊

## Project Overview
This project analyzes delivery performance using invoice data.

## Tools Used
- Excel
- Python
- Pandas
- Data Visualization

## Key Insights
The analysis focuses on:

1. Invoice receive time (9AM–6PM vs After 6PM)
2. Average parts quantity per invoice
3. Delivery type distribution (Bike vs Bus)
4. Delivery time performance

## Goal
To identify patterns in delivery operations and improve logistics efficiency.
